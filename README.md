# 数模实验环境与C程序运行说明

本项目在 **Linux 环境（WSL 或虚拟机）** 下进行操作系统实验

---

## 📌 环境准备
1. **Linux 环境**  
   - 你可以使用 **WSL（Windows Subsystem for Linux）** 或者安装 **Linux 虚拟机（如 Ubuntu）**（建议使用Ubuntu22.04，我在更高版本的运行会出问题原因布吉岛）
   - 确认已经安装**gcc,g++**和**qemu** 

2. **代码编辑器**  
   可使用 `emacs` 或 `vim` 进行代码编写，`vscode`也是极好用的工具

---
